# Frontend

# React + Vite

This is the frontend part of a ride sharing application built with React.
This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Features

Currently, two official plugins are available:

- User authentication and authorization
- User registration and login
- User profile management
- Ride sharing and management
- Captain profile management
- Ride request and management
- Ride status updates

## Technologies Used

- React
- React Router
- React Context API
- React Hooks
- Vite
- ESLint
- Prettier

## How to Run

1. Clone the repository
2. navigate to the frontend directory
3. run `npm install` to install dependencies
4. run `npm run dev` to start the development server

## How to Build

1. run `npm run build` to build the application
2. the built application will be available in the `dist` directory

## How to Test

1. run `npm run test` to run unit tests
2. run `npm run test:coverage` to run unit tests with coverage reports

## How to Lint

1. run `npm run lint` to run ESLint on the codebase
2. run `npm run lint:fix` to run ESLint with automatic fixing of errors

## How to Format Code

1. run `npm run format` to run Prettier on the codebase
