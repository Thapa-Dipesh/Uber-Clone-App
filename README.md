# React Ride Express Full Stack App

This is a full stack application built using React, Express, and MongoDB.

## Tools Used

- Frontend: React, conextAPI, React Router
- Backend: Express, MongoDB, Mongoose
- Database: MongoDB
- Authentication: JWT
- Styling: CSS, Tailwind CSS

## Folder Structure

The project is organized into the following folders:

- `frontend`: Contains the React application
  - `src`: Contains the source code for the React application
  - `public`: Contains static assets for the React application
- `backend`: Contains the Express application
  - `controllers`: Contains the controller functions for the Express application
  - `models`: Contains the Mongoose models for the Express application
  - `routes`: Contains the route definitions for the Express application
  - `services`: Contains the service functions for the Express application

## Running the Application

To run the application, navigate to the `frontend` and `backend` folders and run the following commands:

- `frontend`: `npm start`
- `backend`: `npm start`

The application will be available at `http://localhost:5173` and the API will be available at `http://localhost:3000`.
